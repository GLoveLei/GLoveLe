mac系统中进入mysql遇到Access denied for user 'root'@'localhost'

2019-03-20

Access denied for user ‘root’@’localhost’ (using password: YES)

1)Stop mysql (Kill mysql process or run following command) 停止mysql服务

sudo /usr/local/mysql/support-files/mysql.server stop
  1. Start it in safe mode 用安全模式进入mysql 这一步可以看成是不输入密码就进入mysql命令行
sudo mysqld_safe --skip-grant-tables
  1. Open another terminal and run the following command (Keep last terminal open) 在另一个窗口中进入mysql
mysql -u root
  1. Run the following command with suitable new password on the mysql console 修改密码
UPDATE mysql.user SET Password=PASSWORD('password') WHERE User='root';
  1. 刷新一下缓存
    FLUSH PRIVILEGES;
